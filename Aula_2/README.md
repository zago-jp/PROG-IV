# Fã Clube das Winx

Este é um projeto simples feito em HTML com o objetivo de praticar a criação de páginas web.

A página foi criada recriando um site já existente, apenas para fins de estudo.

## O que contém

- Tabela com personagens das Winx
- Lista de transformações
- Lista de inimigos
- Link para uma wiki externa
- Formulário simples de cadastro

## Tecnologias utilizadas

- HTML
- CSS inline

## Objetivo

Praticar conceitos básicos de:
- Estrutura HTML
- Tabelas
- Listas
- Formulários
- Imagens e links
