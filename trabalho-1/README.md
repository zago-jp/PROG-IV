# 🧭 Hunter x Hunter Wiki

Uma wiki interativa sobre o universo de **Hunter x Hunter (2011)**, desenvolvida utilizando apenas **HTML, CSS e JavaScript puro**.

---

## 📌 Sobre o Projeto

Este projeto apresenta:

- Personagens principais e secundários
- Arcos da história
- Sistema de Nen (técnicas, tipos e conceitos)
- Navegação dinâmica entre páginas
- Conteúdo estruturado via JSON

A proposta é simular uma **enciclopédia digital** do anime, com foco em organização, clareza e interatividade.

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- JSON (para dados)

## 🚀 Como Executar

1. Abra o projeto no VS Code
2. Clique com o botão direito em `index.html`
3. Selecione **Open with Live Server**

Ou abra diretamente no navegador (algumas funções podem depender do Live Server por causa do `fetch`).
